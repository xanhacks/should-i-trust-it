Snapchat

Snapchat est une application de messagerie instantané disponible sur IOS/android

Exodus Privacy
0

D'après Exodus Privacy, l'application Snapchat contient 3 trackers, 2 de Google (Google Ads, Google Firebase Analytics) ainsi que Mapbox

Politique de confidentialité
0

Snapchat collecte les données suivantes : les contacts des utilisateurs, tout les messages / snaps, 'heure et la date de vos échanges, le nombre de messages que vous échangez avec vos amis, les amis avec lesquels vous échangez le plus de messages, ainsi que vos interactions avec ces messages (telles que le moment où vous avez ouvert un message ou pris une capture d'écran), votre localisation et bien d'autres, ces données sont revendues à des fins commerciales...

Chiffrement utilisé
0

Snapchat ne chiffre pas de bout en bout vos messages

Open-Source
0

Ni le client ni le serveur de Snapchat n'est Open-Source

Réputation de l'entreprise
0

L'application Snapchat a été épinglé pour un détournement de données personnelles, Snapchat a aussi été piraté en 2014.
